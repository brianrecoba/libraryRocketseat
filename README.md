# Biblioteca Rocketseat

Este projeto é uma aplicação web simples para gerenciar uma biblioteca pessoal. O usuário pode adicionar livros, filtrar por título ou autor, e marcar o status de leitura.

## Funcionalidades

- Adicionar livros com título, autor, status, avaliação e comentários
- Filtrar livros por título ou autor
- Filtrar por status (todos, lidos, não lidos)
- Interface responsiva com modal para cadastro

## Estrutura do Projeto

```
index.html          # Página principal
assets/              # Imagens e ícones
css/                 # Estilos CSS
js/                  # Scripts JavaScript
```

## Como usar

1. Abra `index.html` em um navegador moderno.
2. Utilize o botão "Adicionar Livro" para inserir novos registros.
3. Use a barra de busca e os filtros para localizar livros.

## Tecnologias

- HTML5
- CSS3
- JavaScript (Vanilla)
